Office 365 IDを利用の場合 Office 365 ID 【5】


【2】 マスターIDで 「V-CUBE One コントロールパネル」 にログイン
1.Oneのトップページを表示し、画面下にある「V-CUBE One コントロールパネルへ」をクリックする。
Oneログイン画面


2.「V-CUBE Oneコントロールパネル（管理者用）」の右下にある「マスターIDでログイン」をクリックする。
V-CUBE One コントロールパネル（管理者用）


3.マスターIDとパスワードを入力し、「ログイン」を押す。
マスターID ログイン画面


4. 「V-CUBE One コントロールパネル」 が表示される。
V-CUBE One コントロールパネル画面




【3】V-CUBE ID （管理者） で 「V-CUBE One コントロールパネル」 にログイン
1.Oneのトップページを表示し、画面下にある「V-CUBE One コントロールパネルへ」をクリックする。
Oneログイン画面


2.V-CUBE IDとパスワードを入力し、「ログイン」を押す。
V-CUBE One コントロールパネル（管理者用）


3.(複数のOneに登録している場合は) ログイン先を選択する。
ログイン先の選択


4. 「V-CUBE One コントロールパネル」 が表示される。
V-CUBE One コントロールパネル画面




【4】 V-CUBE ID （利用者 / 管理者） でOneにログイン
1.Oneのトップページを表示する。

2.V-CUBE IDとパスワードを入力し、「V-CUBE IDでログイン」を押す。
Oneログイン画面


3. (複数のOneに登録している場合は) ログイン先を選択する。
ログイン先の選択


4. ダッシュボードが表示される。
One画面




【5】 Office 365 IDでログイン
1.Oneのトップページを表示し、画面下にある「Office 365でログイン」をクリックする。

2.Office 365のID/パスワードを入力し、「サインイン」ボタンをクリックする。
Office 365 ログイン画面


3. ダッシュボードが表示される。
One画面
